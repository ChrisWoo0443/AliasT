    Replace the hardcoded `suggest()` with:
    ```rust
    pub mod history;

    use history::HistoryStore;

    pub fn suggest(store: &HistoryStore, buffer: &str) -> Option<String> {
        if buffer.is_empty() {
            return None;
        }
        match store.suggest_prefix(buffer) {
            Ok(Some(full_command)) => {
                // Return suffix only -- strip the prefix the user already typed
                let suffix = &full_command[buffer.len()..];
                if suffix.is_empty() {
                    None  // Exact match, nothing to suggest
                } else {
                    Some(suffix.to_string())
                }
            }
            _ => None,
        }
    }
    ```

    The function now takes `&HistoryStore` as its first argument. This is the new public API contract that the daemon connection handler (Plan 02) will use.
