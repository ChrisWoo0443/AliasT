    1. Create `crates/alias-daemon/src/connection.rs`:

    `pub async fn handle_connection(stream: tokio::net::UnixStream, cancel_token: tokio_util::sync::CancellationToken) -> anyhow::Result<()>`:
    - Split stream into reader/writer with `stream.into_split()`
    - Wrap reader in `tokio::io::BufReader`, use `.lines()` for NDJSON framing
    - Loop: read line, parse as `alias_protocol::Request` via `serde_json::from_str`
    - Match on request type:
      - `Request::Complete { id, buf, cur }` -> call `alias_core::suggest(&buf)`, if Some(text) return `Response::Suggestion { id, text }`, if None return `Response::Suggestion { id, text: String::new() }`
      - `Request::Ping { id }` -> return `Response::Pong { id, v: env!("CARGO_PKG_VERSION").to_string() }`
    - If JSON parse fails -> return `Response::Error { id: "unknown".to_string(), msg: error_message }`
    - Serialize response with `serde_json::to_string`, write with newline suffix
    - Use `tokio::select!` with `cancel_token.cancelled()` to exit on shutdown
    - Log connection open/close with `tracing::debug!`

    2. Create `crates/alias-daemon/src/server.rs`:

    `pub async fn run_server(socket_path: &Path, cancel_token: tokio_util::sync::CancellationToken) -> anyhow::Result<()>`:
    - Call `alias_daemon::lifecycle::cleanup_stale_socket(socket_path)?`
    - Bind `tokio::net::UnixListener` at socket_path
    - `tracing::info!("Listening on {:?}", socket_path)`
    - Accept loop using `tokio::select!` with cancel_token:
      - On accept: `tokio::spawn(handle_connection(stream, cancel_token.child_token()))`
      - On cancel: break, log shutdown
    - After loop exits, call `lifecycle::remove_socket(socket_path)`

    3. Update `crates/alias-daemon/src/main.rs`:
    - Add `mod server; mod connection; mod lifecycle;`
    - In `Commands::Start` arm:
      - Resolve socket_path from arg or `lifecycle::default_socket_path()`
      - Create `CancellationToken`
      - Spawn `server::run_server` task
      - Wait for SIGINT or SIGTERM via `tokio::signal::ctrl_c()` and `tokio::signal::unix::signal(SignalKind::terminate())`
      - On signal: cancel token, await server task, log clean exit
    - In `Commands::Stop` arm: print "stop not yet implemented" (will connect to daemon and send shutdown in future)
    - In `Commands::Status` arm: try connecting to socket path, print running/not running
    - Import `tokio_util::sync::CancellationToken` and `tokio::signal::unix::{signal, SignalKind}`

    4. Write integration tests in `crates/alias-daemon/tests/server_tests.rs`:
    - Each test creates a temp directory, starts a server with `run_server` on a temp socket path, connects a `tokio::net::UnixStream` client, sends NDJSON, reads response, asserts.
    - Use `tokio::time::timeout` to prevent hanging tests (5 second timeout per test).
    - Cancel the server after each test via the CancellationToken.
