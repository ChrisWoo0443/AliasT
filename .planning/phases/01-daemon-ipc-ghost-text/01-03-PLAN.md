<interfaces>
<!-- NDJSON protocol format the plugin must produce and consume -->

Request (plugin sends to daemon):
```json
{"id":"r1","type":"complete","buf":"git ch","cur":6}
{"id":"r0","type":"ping"}
```

Response (daemon sends to plugin):
```json
{"id":"r1","type":"suggestion","text":"eckout main"}
{"id":"r0","type":"pong","v":"0.1.0"}
{"id":"r1","type":"error","msg":"not found"}
```

Socket path: $XDG_RUNTIME_DIR/alias/alias.sock or /tmp/alias-$UID/alias/alias.sock

Key ZLE patterns (from research):
- POSTDISPLAY: ghost text variable (text after cursor, not part of buffer)
- region_highlight: array of "{start} {end} {style} memo={tag}" entries
- zle -F $fd callback: register async fd reader
- zle -R: force redisplay
- add-zle-hook-widget: non-conflicting hook registration
</interfaces>
</context>

    Create `plugin/alias.plugin.zsh` implementing the complete plugin. Follow the skeleton from RESEARCH.md Pattern 4, 5, 6. The file must implement these sections in order:

    **1. Guard and module loading:**
    - `(( ${+_ALIAS_LOADED} )) && return` to prevent double-loading
    - `typeset -g _ALIAS_LOADED=1`
    - `zmodload zsh/net/socket || { print "alias: zsh/net/socket module required" >&2; return 1; }`
    - `autoload -Uz add-zle-hook-widget`

    **2. State variables:**
    - `typeset -g _ALIAS_FD=""` (socket file descriptor)
    - `typeset -g _ALIAS_REQ_ID=0` (monotonic request counter)
    - `typeset -g _ALIAS_HIGHLIGHT_ENTRY=""` (tracked region_highlight entry)
    - `typeset -g _ALIAS_LAST_BUFFER=""` (buffer change detection)
    - `typeset -g _ALIAS_SOCKET_PATH="${XDG_RUNTIME_DIR:-/tmp/alias-$UID}/alias/alias.sock"` (per D-04)

    **3. Connection management (lazy init -- no I/O at plugin load time):**
    - `_alias_connect()`: if `_ALIAS_FD` is already set, return 0. Otherwise `zsocket "$_ALIAS_SOCKET_PATH" 2>/dev/null || return 1`. Set `_ALIAS_FD=$REPLY`. Register `zle -F $_ALIAS_FD _alias_handle_response`. Return 0.
    - `_alias_disconnect()`: if `_ALIAS_FD` set, `zle -F $_ALIAS_FD` to unregister, `exec {_ALIAS_FD}>&-` to close fd, set `_ALIAS_FD=""`.
    - `_alias_reconnect()`: call `_alias_disconnect`. Attempt daemon respawn per D-02: `command alias-daemon start &>/dev/null &!`. (Next keystroke retries connection.)

    **4. Ghost text rendering (per D-06, D-07):**
    - `_alias_show_ghost()`: takes `$1` as suggestion text. Calls `_alias_clear_ghost` first. Sets `POSTDISPLAY="$1"`. If `$#POSTDISPLAY > 0`: compute `start=$#BUFFER`, `end=$(( start + $#POSTDISPLAY ))`, `style="${ALIAS_SUGGESTION_HIGHLIGHT:-fg=8}"` (D-06, D-07). Set `_ALIAS_HIGHLIGHT_ENTRY="${start} ${end} ${style} memo=alias"`. Append to `region_highlight+=("$_ALIAS_HIGHLIGHT_ENTRY")`.
    - `_alias_clear_ghost()`: if `_ALIAS_HIGHLIGHT_ENTRY` is set, remove from region_highlight with `region_highlight=("${(@)region_highlight:#$_ALIAS_HIGHLIGHT_ENTRY}")`, set `_ALIAS_HIGHLIGHT_ENTRY=""`. Set `POSTDISPLAY=""`.

    **5. IPC -- send requests and handle responses:**
    - `_alias_send_request()`: call `_alias_connect || return`. Increment `(( _ALIAS_REQ_ID++ ))`. Escape `$BUFFER` for JSON: replace `\` with `\\` and `"` with `\"`. Construct JSON: `"{\"id\":\"r${_ALIAS_REQ_ID}\",\"type\":\"complete\",\"buf\":\"${escaped_buffer}\",\"cur\":${CURSOR}}"`. Send with `print -u $_ALIAS_FD "$msg" 2>/dev/null || { _alias_reconnect; return; }`.
    - `_alias_handle_response()`: receives `$1` (fd) and `$2` (error flag). If `$2` is non-empty or `read -r -u $fd line` fails: `zle -F $fd; _ALIAS_FD=""; return`. Parse response type: check if `$line` contains `"type":"suggestion"`. If so, extract `text` field with pattern matching: `local text="${line##*\"text\":\"}"; text="${text%%\"*}"`. Extract `id` field: `local resp_id="${line##*\"id\":\"}"; resp_id="${resp_id%%\"*}"`. If `$resp_id` equals `"r${_ALIAS_REQ_ID}"` (staleness check), call `_alias_show_ghost "$text"` and `zle -R`.

    **6. Widget wrappers (minimal per D-08):**
    - Save original: `zle -A self-insert _alias_orig_self_insert`
    - `_alias_self_insert_wrapper()`: `zle _alias_orig_self_insert "$@"` then `_alias_send_request`
    - `zle -N self-insert _alias_self_insert_wrapper`
    - Save original: `zle -A accept-line _alias_orig_accept_line`
    - `_alias_accept_line_wrapper()`: `_alias_clear_ghost` then `zle _alias_orig_accept_line "$@"`
    - `zle -N accept-line _alias_accept_line_wrapper`

    **7. Hook registration (non-conflicting per D-08):**
    - `_alias_line_pre_redraw()`: if `$BUFFER != $_ALIAS_LAST_BUFFER`: set `_ALIAS_LAST_BUFFER="$BUFFER"`. If buffer is empty, call `_alias_clear_ghost`.
    - `add-zle-hook-widget zle-line-pre-redraw _alias_line_pre_redraw`

    **8. Compatibility detection:**
    - Check for zsh-autosuggestions: `if (( ${+_ZSH_AUTOSUGGEST_HIGHLIGHT_STYLE} )) || [[ -n "${functions[_zsh_autosuggest_suggest]}" ]]; then print "alias: zsh-autosuggestions detected. Ghost text may conflict." >&2; fi`
